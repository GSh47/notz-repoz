
Given a function 
$$
Adx+Bdy+Cdz=0
$$
is exact if 
$$
\begin{matrix}
\frac{ \partial A }{ \partial y } =\frac{ \partial B }{ \partial x }  \\
\frac{ \partial A }{ \partial z } =\frac{ \partial C }{ \partial x }  \\
\frac{ \partial B }{ \partial z } =\frac{ \partial C }{ \partial y } 
\end{matrix}
$$
this is the condition for exactness. Now go !
